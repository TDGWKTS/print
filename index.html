📄 Print Inspection Records

Export CSV data, then copy into your Excel template
📋 How to print (preserving all formatting):
  1. Click Export Data as CSV to download the inspection data
  2. Open your Excel template (the one with all your formatting)
  3. Go to the DataImport sheet (or create one)
  4. Select cell A2
  5. Go to Data → From Text/CSV and select the downloaded CSV file
  6. Click Load – data will appear in rows 2-6
  7. The Print sheet will automatically update via your formulas
  8. Print as usual – all formatting preserved!
💡 Why this works: Your Excel template is never modified.
The CSV file is imported directly into Excel's DataImport sheet.
100% of your formatting (fonts, colors, merged cells) is preserved.
