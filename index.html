📄 Printing Instructions

  1. Download Template (First Time Only) – Click Download Template to get the master Excel template for the selected station.
  2. Download Inspection Data – Click Download Inspection Data to get the inspection records for the selected station and date.
  3. Open Both Files – Open the downloaded template and the inspection data file.
  4. Copy Data – In the inspection data file, select all data cells from A2 to the last row/column, then press Ctrl+C.
  5. Paste to Print Worksheet – In the template file, go to the Print worksheet, select cell A6, then press Ctrl+V.
  6. ✓ Print – Your data is now in the Print sheet. All formulas will auto-calculate. Review and print.
