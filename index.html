📄 Print Inspection Records

West Kowloon Transfer Station (WKTS)
1
Download Template (First Time Only)
Click Download Excel Template to get the master template with all formatting and formulas.
2
Download Inspection Data
Click Download Inspection Data to get the inspection records for the selected date.
3
Open Both Excel Files
Open the Template file and the Inspection Data file side by side.
4
Copy Data
In the Inspection Data file, select all data cells (A2 to last row/column), press Ctrl+C.
5
Paste to Print Worksheet
In the Template file, go to the Print worksheet, select cell A6, press Ctrl+V.
✓
Print
Your data is now in the Print sheet. All formulas will auto-calculate. Review and print.
💡 Important: Your Excel template is never modified by any server.
You simply copy data from the downloaded file and paste into your Print sheet row 6.
All your formulas (checkmarks, calculations) and formatting remain intact.
