📄 Print Inspection Records

West Kowloon Transfer Station (WKTS)
1
Download Inspection Data
Click Download Inspection Data to get the inspection records for the selected date.
2
Open Your Excel Template
Open your existing WKTS Daily Inspection Tool(Template) V6.xlsx file.
3
Open Downloaded File
Open the downloaded Excel file (wkts_YYYY-MM-DD_inspection_data.xlsx).
4
Copy Data
In the downloaded file, select all data cells from A2 to the last row/column, then press Ctrl+C.
5
Paste to Print Worksheet
In your template file, go to the Print worksheet, select cell A6, then press Ctrl+V.
✓
Print
Your data is now in the Print sheet. All formulas will auto-calculate. Review and print.
💡 Important: The downloaded Excel file has the EXACT column headers matching your Print sheet.
Simply copy cells A2 to the end and paste into your Print sheet at row 6.
