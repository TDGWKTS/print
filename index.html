📄 Print Inspection Records

Download CSV data, then import into your Excel template
📋 How to print (preserving all formatting):
  1. Select Station and Date
  2. Click Download CSV Data to get inspection data
  3. Select your Excel Template file (the one with all your formatting)
  4. Click Import CSV into Template to merge data
  5. The merged Excel file will download automatically
  6. Open and print – all formatting preserved!
💡 Why this works: Your Excel template is never modified by any server.
The CSV import happens locally in your browser using Excel's built-in capabilities.
100% of your formatting (fonts, colors, merged cells) is preserved.
